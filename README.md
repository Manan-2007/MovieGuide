Project Title: MovieGuide – A Dynamic Movie Review Application
Overview
MovieGuide is a responsive web application designed to provide users with instant access to movie information, reviews, and ratings. Built entirely using HTML5, CSS3, and Vanilla JavaScript, this project demonstrates the practical application of frontend web development concepts, including API integration, DOM manipulation, and multi-page navigation.

Key Features

Real-Time Search Engine: The core functionality is powered by the OMDb API. Users can search for any movie title to dynamically fetch and display real-time data, including high-quality posters, plot summaries, cast lists, and IMDb ratings.

Multi-Page Navigation: Unlike simple single-page sites, MovieGuide features a complete navigation architecture connecting five distinct pages: Search (Home), Trending, About Us, Login, and Sign Up.

Curated Content: The "Trending Now" page utilizes modern CSS Grid and Flexbox layouts to showcase "Editor's Pick" movies with a responsive card design.

User Simulation: The application mimics a real-world platform with a functional Login and Sign-Up flow. This includes form validation (ensuring fields are not empty) and JavaScript-based redirection logic to guide the user through the authentication process.

Modern UI/UX: The interface features a consistent dark-themed aesthetic (using the #1c1917 and #ffb92a color palette), ensuring a visually engaging and mobile-responsive experience across all devices.

Technologies Used

Frontend: HTML5, CSS3 (Grid/Flexbox), JavaScript (ES6+)

Data Source: OMDb API (Open Movie Database)

Tools: Visual Studio Code, Git

This project effectively bridges the gap between static web design and dynamic, interactive web applications.
